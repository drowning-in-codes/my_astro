# Leetcode周赛

Q1给你一个整数 n。

返回将所有从 [1, n]（包含两端）范围内的整数以 标准 数字格式书写时所用到的 逗号总数。

在 标准 格式中：

从右边开始，每 三位 数字后插入一个逗号。
位数 少于四位 的数字不包含逗号。

```c++
class Solution {
public:
    int countCommas(int n) {
        long long res{};
        for(long long i =1000;i<=n;i*=1000) {
            res += n-i+1;
        }
        return res;
    }
};
```
